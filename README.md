## Deploy python environment for Ubuntu
```
sudo apt-get update
sudo apt-get install python3
sudo apt-get install python3-pip
sudo apt-get install python3-venv
python3 -m venv env
```
## Activate and deactivate env
```
source tutorial-env/bin/activate
deactivate
```
